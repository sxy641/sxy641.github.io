set——集合

引文：

集合类似于数学中的集合， 其中每个元素只能出现一次， 与py相比不同的是， py使用set()返回的是一个无序的元组， 而在C++中set的string已经重载了’<’运算符， 插入的字符串会以字典序进行排序。

特征： 被插入进set的元素都会在set里进行自动排序和去重。
